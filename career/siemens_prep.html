1. PPCL Scripting (Powers Process Control Language)

PPCL is a line-oriented, procedural language used for custom sequences in Siemens field panels.

Core Syntax Commands

Command Usage
SET Set value to point (e.g., SET(72, ROOM.TEMP.SP))
IF/THEN Conditional logic (e.g., IF (VAL > 75) THEN ON(FAN))
WAIT Pause execution (e.g., WAIT(300))
GOTO Jump to line (e.g., GOTO 100)

Sample: Fan Control Logic

100 IF (ROOM.TEMP > 75) THEN ON(FAN.CMD)
110 IF (ROOM.TEMP < 70) THEN OFF(FAN.CMD)
120 GOTO 100

3. Network Protocols

  • BACnet/IP: high-speed Ethernet communication between servers and field panels.
  • BACnet MS/TP: RS-485 twisted pair communication for smaller field devices (TECs/PTECs).
  • LonWorks: Decentralized peer-to-peer protocol (requires gateway for BAS integration).
